Documentação do Projeto

1. Estrutura de Arquivos

  • index.html - Página inicial
  • jogadores/ - Páginas individuais dos jogadores
  • dicas_do_jogo.html - Página de dicas
  • assets/ - Recursos como imagens e estilos

2. Componentes Bootstrap Utilizados

  • Navbar: Menu responsivo com dropdown
  • Cards: Exibição de informações dos jogadores
  • Modal: Popups informativos
  • Offcanvas: Menu lateral para dicas
  • Toast: Notificações e dicas rápidas

3. Escolhas de Design

Layout Responsivo

Uso do sistema de grid do Bootstrap para adaptação em diferentes telas.

Cores e Tema

Esquema escuro (dark theme) para melhor experiência visual.

Componentes Interativos

Uso de modais e offcanvas para melhor organização do conteúdo.

4. Funcionalidades Principais

  • Perfil detalhado dos jogadores
  • Sistema de dicas para iniciantes e avançados
  • Timeline de conquistas
  • Histórico de times
